How To
------

## Serve
Run:

```
python -m SimpleHTTPServer
```

Then visit your browser on:

+ `http://localhost:8000/ABC.html`
+ `http://localhost:8000/CBA-with-auto-width.html`
+ `http://localhost:8000/BAC.html`

## View Diff

Please `diff` none.html and other files.
